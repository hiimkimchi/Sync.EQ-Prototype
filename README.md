# Sync.EQ - Equalize Your Connections
Quarter Long Project for CPSC 5240

## Prerequisites needed
- NodeJS

## To run the Node/Express Backend 
```
cd backend
npm install
node server.js
```

## env vars
```
MONGO_URL
```
### NOTE: your .env file should be in /backend

## To run the Angular Frontend
### Generate config.json
```
cd frontend
npm run createConfig
```

### Run App
```
cd frontend
npm install
npm run start
```

## env vars
```
API_URL
```
### NOTE: your .env file should be in the /frontend
